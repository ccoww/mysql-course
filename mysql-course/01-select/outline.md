# 第1课：SELECT 基础查询

**主题**：从单表中查询数据
**时长**：90 分钟
**前置要求**：无

## 时间分配

| 时段 | 内容 | 时长 |
|------|------|------|
| 0-10min | 导入：数据库和表的概念 | 10min |
| 10-20min | 认识电商数据库 | 10min |
| 20-40min | SELECT 基础语法 | 20min |
| 40-60min | WHERE 条件过滤 | 20min |
| 60-75min | 排序与分页 | 15min |
| 75-90min | 课堂练习 | 15min |

## 教学目标

- 理解表、行、列、主键的概念
- 掌握 SELECT 查询语法
- 能用 WHERE 过滤数据
- 能用 ORDER BY 排序、LIMIT 分页

## 不在本课范围

- 多表 JOIN（第2课）
- 聚合统计（第3课）
