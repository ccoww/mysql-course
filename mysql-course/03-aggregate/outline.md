# 第3课：聚合统计与分组

**主题**：对数据进行汇总分析
**时长**：90 分钟
**前置要求**：第1、2课

## 时间分配

| 时段 | 内容 | 时长 |
|------|------|------|
| 0-15min | 聚合函数简介 | 15min |
| 15-35min | GROUP BY 分组 | 20min |
| 35-50min | HAVING 分组过滤 | 15min |
| 50-70min | 综合查询练习 | 20min |
| 70-90min | 课堂练习 | 20min |

## 教学目标

- 掌握 COUNT/SUM/AVG/MAX/MIN 用法
- 能用 GROUP BY 进行分组统计
- 能用 HAVING 过滤分组结果
