# 第2课：JOIN 多表查询

**主题**：从多张关联表中查询数据
**时长**：90 分钟
**前置要求**：第1课 SELECT 基础

## 时间分配

| 时段 | 内容 | 时长 |
|------|------|------|
| 0-15min | 为什么要多表查询 | 15min |
| 15-35min | INNER JOIN 内连接 | 20min |
| 35-55min | LEFT/RIGHT JOIN 外连接 | 20min |
| 55-70min | 多表 JOIN 与子查询 | 15min |
| 70-90min | 课堂练习 | 20min |

## 教学目标

- 理解表与表之间的关系（主键、外键）
- 掌握 INNER JOIN / LEFT JOIN 语法
- 能进行多表关联查询
